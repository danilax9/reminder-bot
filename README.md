# Telegram Бот для напоминаний (Aiogram v3)

Бот позволяет создавать напоминания, анализируя текст сообщений на русском языке.

## Возможности
- Распознает даты и время в свободной форме (завтра, 2 марта, в 15:00 и т.д.)
- Поддерживает относительное время (через 10 минут, через 2 часа)
- **Повторяющиеся напоминания**:
  - Каждый день (ежедневно)
  - Каждую неделю (каждый понедельник, каждую пятницу и т.д.)
- Умный выбор "сегодня/завтра" для времени без указания даты
- Если время не указано, устанавливает на 09:00

## Примеры команд
- `каждый день пить витамины в 9:00` -> напоминание ежедневно в 09:00
- `каждый понедельник планерка в 10:00` -> напоминание раз в неделю
- `вынести мусор через 10 минут` -> разовое напоминание
- `покормить кота в 18:00` -> напомнит сегодня (если еще нет 18:00) или завтра
- `2 марта в 15.00 пойти к врачу` -> разовое напоминание

## Установка и запуск

1. Склонируйте репозиторий.
2. Установите зависимости:
   ```bash
   pip install -r requirements.txt
   ```
3. Создайте файл `.env` на основе `.env.example` и добавьте туда ваш `BOT_TOKEN`:
   ```bash
   cp .env.example .env
   ```
4. Запустите бота:
   ```bash
   python main.py
   ```

## Используемые библиотеки
- `aiogram` (v3.x) - для работы с Telegram API
- `apscheduler` - для планирования отправки сообщений
- `dateparser` - для парсинга дат и времени из текста
- `python-dotenv` - для работы с переменными окружения
